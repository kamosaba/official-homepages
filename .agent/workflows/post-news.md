---
description: ニュース記事の投稿方法
---

かも鯖公式サイトに新しいニュースを投稿する手順は以下の通りです。

### 1. Markdownファイルの作成
`src/content/news` フォルダの中に、新しい `.md` ファイルを作成します。
ファイル名は `2026-01-11-new-event.md` のように、日付を入れると管理しやすくなります。

### 2. フロントマターの設定
ファイルの先頭に、以下のような設定（フロントマター）を記述してください。

```markdown
---
title: '記事のタイトル'
date: '2026-01-11'
description: '一覧ページに表示される短い説明文'
---
```

### 3. 本文の執筆
フロントマターの下に、Markdown形式で本文を記述します。
見出し（##）、太字（**）、箇条書き（- ）などが使用可能です。

### 4. 保存とデプロイ
ファイルを保存し、変更を Git リポジトリに **Push** してください。
Cloudflare Pages が自動的にビルドを開始し、数分ほどで公式サイトに反映されます。

詳細ページ（/news/ファイル名/）も自動的に生成されます。
