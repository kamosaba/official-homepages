# JAVA版の場合

## 参加情報 <a href="#join-address" id="join-address"></a>

```markup
play.kamosaba.net
```
```markup
かもさば.net
```
上記どちらかで参加することができます🌟

## 参加方法

1. Minecraftを起動しましょう

<figure><img src="public/images/docs/assets/image (2).png" alt="test"><figcaption><p>マインクラフトの起動後の画面</p></figcaption></figure>

2. マルチプレイを押すと、下記画像のようになります。

<figure><img src="../.gitbook/assets/image (1) (1).png" alt=""><figcaption></figcaption></figure>

3. サーバーを追加/ダイレクト接続を押すと、下記画面になります。 参加情報に書いてある`play.kamosaba.net`または、`かもさば.net`をサーバーアドレスの欄に書きましょう

<figure><img src="../.gitbook/assets/スクリーンショット 2024-09-01 223915.png" alt=""><figcaption></figcaption></figure>

4. **保存**を押し、接続ボタンを押すと接続することができます。/ **サーバーに接続** を押すと、接続できます！

サーバーに入って楽しみましょう！
