# 🎮 Java版 参加ガイド

「かも鯖」へ参加するための手順を、初心者の方にもわかりやすくステップ形式で解説します。✨

### 📝 準備するもの
* **Minecraft: Java Edition**（PC版）
* インターネット接続環境

### 🚀 参加のステップ

1. **マインクラフトを起動する**
   ランチャーからJava Editionを起動し、メインメニューを表示します。

2. **「マルチプレイ」を選択する**
   メニュー画面にある「マルチプレイ」ボタンをクリックします。

3. **「サーバーを追加」をクリックする**
   画面右下にある「サーバーを追加」ボタンを押します。

4. **サーバー情報を入力する**
   以下の情報を入力枠にコピー＆ペーストしてください。
   * **サーバー名**: かも鯖（自分がわかりやすい名前でOK）
   * **サーバーアドレス**: 以下のいずれかを入力
     1. `play.kamosaba.net`
     2. `かもさば.net`

5. **「完了」を押して接続する**
   入力が終わったら「完了」を押し、サーバー一覧に追加された「かも鯖」をダブルクリック（または選択して「サーバーに参加」）すれば、冒険の始まりです！🦆

### 💡 うまく接続できない場合
* **アドレスの入力ミス**: 余計なスペース（空白）が入っていないか確認してください。
* **バージョン**: サーバーが対応している最新のバージョンで起動しているか確認してください。
