# 📖 かも鯖 公式ドキュメントへようこそ

かも鯖を楽しく、快適にプレイするための情報をまとめた公式ドキュメントです。✨

### 📝 掲載されている内容
ドキュメントでは、主に以下の情報を確認することができます。

* **🎮 参加方法**: サーバーへの接続手順や必要な設定について
* **⚙️ 機能紹介**: サーバー独自のシステムや便利なコマンドの使い方
* **🎉 イベント情報**: 開催中・開催予定のイベントに関する詳細ルール

### 🔍 困ったときは
プレイ中に「使い方がわからない」「ルールを確認したい」と思ったときは、まず最初にこの**ドキュメント**を確認してみましょう。🔍
